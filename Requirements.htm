Requirements

By Niall Stack

 

·         A student must log in with their credentials in order to view the ‘whiteboard i.e. Status page board and access all other application features.

·         A student can only view the statuses posted on the ‘whiteboard by other students in hers/his course of study. 

·         A student can view another student’s profile, as long as they are in the same course of study.

·         A student may view all statuses that they have posted.

·         A student may not provide confidential or other proprietary information.

·         A student may have only one profile.  At any time the student may change his group or school, if needed.

·         A student can only be between 10 and 20 years old.

·         When a student views his/hers friends list, he/she can only view the friends that he/she is connected to or friends with.

·         Whenever a message is posted in a group, all members get a notification while if a message is posted on a school post, they do not.

·         A student’s profile picture must only include the student and it must be a profile shot, passport photo like.

·         Password must be between 4 and 20 characters and must contain at least one uppercase letter and one digit.

·         Students profile picture must be under 1mb in size when uploading.

·         Forename and surname must start with a capital letter.

·         A student must have at least one interest.

·        Each Student can accept any amount of friend requests while only one friend request is accepted by exactly one student.

·        Each student can send any number of messages while each message is sent by one student.

·        Each student may be notified by any amount of alerts while an alert notifies exactly one student.

·        Each student can select one school while a school has any number of students

·        Each student can select one group while a group has any number of students

·        Each student can select any number of interests and each interest can have any number of students.
