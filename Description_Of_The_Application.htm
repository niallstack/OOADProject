Overview

Of the Project

 

This is a document related to the ‘Playground’ application we chose to build.  This application is designed for students and staff which enables both to interact with the application in different ways.  The application enables students to communicate with each other using private messages, posts on the wall which the scope can be limited to either the class the student belongs to or the school.  Each student has their own profile which contains their profile photo and some basic information such as student number, forename, surname etc.  Every student can view each other’s profile and their list of friends.

As for the staff, mainly the School Office Administrator (SOA) in this case, can register new students on this application, revoke permission for a student, amend a student’s information and also view every student’s profile.  But the SOA does not have their own profile nor any friends on the application.
