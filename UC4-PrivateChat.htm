Use Case Name

Private Chatting

Author: Quentin

Use Case Id

UC4

Date: 25/11/2015

Priority

4

Source

Requirements

Primary Business Actor

Student

Other Participating Actors

Another Student

Description

As a student I can send a private message to a friend of mine and can expect an answer.

Preconditions

Both students must be registered and friends.

Trigger

The student clicks the view notification function.

Typical Scenario

Actor Action

System Response

 

Step 1: The student choose one of his friend in his friend list.

 

Step 3: The student write the message inside the chatBox and send it.

Step 2: The system display a chat box.

 

Step 4: The system display the new message in the chat box.

 

Step 5: The system send a notification to the friend.

 

Alternate Scenarios

Actor Action

System Response

Required field not entered

Step 1:The student send a message without writing anything

 

 

Step 2:The system doesnt do anything.

 

Conclusions

The message has successfully been sent.

Post conditions

Business Rules

You have to be friend with the student you want to chat.

Implementation Constraints
